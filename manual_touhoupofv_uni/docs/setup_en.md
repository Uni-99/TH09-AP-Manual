# Touhou PoFV Archipelago Manual Randomizer Setup Guide

## Required Software

- The [Archipelago Launcher](https://archipelago.gg/)
- Touhou PoFV Game ([Steam](https://store.steampowered.com/app/1420810/Touhou_Kaeizuka__Phantasmagoria_of_Flower_View/))
  - A complete save file with all characters unlocked is required
    - If you don't have all characters unlocked, you can use the unlock code by heading to the result screen and enter "__DDDSSSSQRR__"

## Optional Software

- [Universal Tracker](https://github.com/FarisTheAncient/Archipelago/releases/tag/Tracker_v0.2.24.1)
  - An apworld that shows you which of your locations are currently in-logic.
- [thprac](https://github.com/touhouworldcup/thprac)
  - A tool for practicing Touhou shmups. This can be used to make opponents invulnerable for better consistency on difficulty, aswell as to immediately kill the player for death link.
  - Only works for Match Mode.

## Installation Procedures

Place the manual_touhoupofv_uni.apworld in the *custom_worlds* folder of your archipelago install.
  - ([Your Archipelago Folder]\custom_worlds)

## Joining a MultiWorld Game

1. Open up the Manual client through *ArchipelagoLauncher.exe*.
2. Set Manual Game ID to "Manual_TouhouPoFV_Uni".
3. Enter the server's ip and port at the top and press the connect button on the top right.
4. In the command field at the bottom, input the slot name you chose in your yaml file and press enter.
